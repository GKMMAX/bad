## 📤 上传步骤（5 分钟）

### 步骤 1：登录/注册 GitHub

1. 访问：https://github.com
2. 如果已有账号，直接登录
3. 如果没有，点击 "Sign up" 创建免费账号（2 分钟）

### 步骤 2：创建新仓库

1. 登录后，点击右上角 **"+"**
2. 选择 **"New repository"**
3. 填写仓库信息：
   - **Repository name**: `BadmintonAndroid`
   - **Description**: `羽毛球场地预订 Android 应用`
   - **Visibility**: 选择 `Public`（推荐）或 `Private`
4. 点击 **"Create repository"** 按钮

### 步骤 3：上传项目文件

**方法 A：网页上传（推荐，最简单）**

1. 在新创建的仓库页面，点击 **"uploading an existing file"**
2. 打开文件资源管理器
3. 进入文件夹：`C:\Users\hp\WorkBuddy\Claw\BadmintonAndroid`
4. 选中所有文件和文件夹：
   - 按 `Ctrl + A` 全选
   - 或拖拽整个 `BadmintonAndroid` 文件夹到浏览器
5. 等待上传完成（可能需要 2-3 分钟）
6. 在页面底部填写提交信息：
   - **Commit message**: `Initial commit`
7. 点击 **"Commit changes"** 绿色按钮

**方法 B：使用 Git 命令（如果有 Git）**

```bash
cd C:\Users\hp\WorkBuddy\Claw\BadmintonAndroid
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/BadmintonAndroid.git
git branch -M main
git push -u origin main
```

### 步骤 4：触发构建

1. 上传完成后，在仓库页面点击 **"Actions"** 标签
2. 找到工作流：**"Build Android APK"**
3. 点击右侧的 **"Run workflow"** 按钮
4. 在弹出的窗口中：
   - **Branch**: 选择 `main`
   - **Build type**: 选择 `debug`
5. 点击 **"Run workflow"** 绿色按钮

### 步骤 5：等待构建完成

- 构建时间：**10-15 分钟**
- 构建状态：
  - ⚪ 灰色：等待中
  - 🟡 黄色：构建中
  - 🟢 绿色：成功
  - 🔴 红色：失败

### 步骤 6：下载 APK

1. 构建完成后，刷新页面
2. 点击最新的构建记录（绿色勾图标）
3. 滚动到页面最底部
4. 在 **"Artifacts"** 部分
5. 找到 `app-debug.apk`
6. 点击下载
