## 构建和运行

### 开发环境要求

- Android Studio Hedgehog (2023.1.1) 或更高
- JDK 17
- Android SDK API 34

### 快速开始

1. **打开项目**
   ```bash
   # 在 Android Studio 中打开 BadmintonAndroid 文件夹
   ```

2. **同步 Gradle**
   - 点击 "Sync Now"
   - 等待同步完成

3. **运行应用**
   - 连接 Android 设备或启动模拟器
   - 点击 Run 按钮 (▶) 或按 Shift + F10

### 构建 APK

#### Debug 版本

```bash
# Windows
gradlew.bat assembleDebug

# APK 位置: app/build/outputs/apk/debug/app-debug.apk
```

#### Release 版本

```bash
# 1. 创建密钥库（首次需要）
keytool -genkey -v -keystore badminton-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias badminton-key

# 2. 构建 Release APK
# Windows
gradlew.bat assembleRelease

# 或通过 Android Studio: Build -> Generate Signed Bundle / APK
```
