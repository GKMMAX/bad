## 🔍 如何找到 JDK 路径

### 快速查找方法：

1. **方法 1：控制面板**
   - 控制面板 → 程序和功能
   - 找到 "Java SE Development Kit" 或 "Eclipse Adoptium"
   - 右键 → 更改 → 查看安装路径

2. **方法 2：文件资源管理器**
   - 打开 `C:\Program Files\`
   - 搜索 "java.exe"
   - 找到 `jdk-17.x.x\bin\java.exe`
   - JDK 路径 = 该文件所在的上两级目录

## ⚠️ 构建过程中可能遇到的问题

### 问题 1：找不到 Java
**解决**：
- 确认 JDK 17 已安装
- 检查 JAVA_HOME 路径是否正确
- 路径末尾不要加 `\bin`

### 问题 2：下载依赖很慢
**解决**：
- 耐心等待（首次构建需要下载约 500MB）
- 使用手机热点可能更快
- 如果超时，尝试 GitHub Actions 在线构建

### 问题 3：构建失败
**解决**：
- 查看错误信息
- 确保网络连接正常
- 尝试删除 `app\build` 文件夹后重新构建
